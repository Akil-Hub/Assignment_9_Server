
# Sports Facility Reservation System - Backend

## Project Overview

This is the backend server for the Sports Facility Reservation System. It handles authentication, facility management, booking operations, and database communication.

The backend is built with Express.js and MongoDB.

## Live Server

- Server URL: https://assignment-9-server-roan.vercel.app/

---

## Features

- REST API for facility management
- Booking management system
- Secure authentication support
- MongoDB database integration
- Environment variable configuration
- CORS enabled for frontend communication
